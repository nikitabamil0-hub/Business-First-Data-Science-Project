# Capstone Project 5: Business-First Data Science Project

## Project Overview
This project follows a Business-First approach to Data Science, where the focus is on understanding business objectives before applying analytical and machine learning techniques. The goal is to solve real-world business challenges through data-driven decision-making while ensuring that technical solutions align with organizational goals.

## Problem Statement
Many data science projects fail because they focus on technology rather than business needs. This project emphasizes identifying business requirements, defining measurable objectives, and developing data-driven solutions that create tangible business value.
